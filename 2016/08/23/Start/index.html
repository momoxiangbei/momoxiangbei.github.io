Start

发表于 2016-08-23   |  

我决定要每天写博客了，就像写日记一样
弄了两天，用hexo+github搭好了自己的博客网站，用的Next主题，看着很简单舒服。
我每次做什么东西，都要给自己个仪式感，给自己个起始点，其实就是找个心理安慰。比如说每月1号，每周一，每天早上，都有一种重新开始，好好做人的感觉。这次也小小的期待着通过自己搭的博客网站，开个好头。

下面就是我的自问自答了
为什么要写博客？
1、提高技术水平
2、每天晚上闲着也是闲着
3、分享精神？
4、最主要的是：我是一条有梦想的咸鱼

更新博客相关的自我约束？
1、工作日必更，法定节假日随意
2、多存几篇，防止懒癌突发，影响进度
3、要保证质量，博客可以很短，但不可很烂
4、博客内容不限，技术、生活、吐槽等等

没什么要说的了，so，开始吧！
